# DanmakuYT

A browser extension built with WXT, React, Radix UI, and Tailwind CSS.

## Quick Start

1. Install dependencies:
   ```bash
   bun install
   ```

2. Start development:
   ```bash
   bun run dev
   ```

3. Build for production:
   ```bash
   bun run build
   ```

## Available Scripts

- `bun run dev` - Start development server with hot reload
- `bun run build` - Build extension for production
- `bun run zip` - Create distribution zip file
- `bun run check` - Type-check TypeScript files

## Bun Commands

Bun is used as the package manager and runtime. Common Bun commands:

```bash
bun install          # Install dependencies
bun add <package>    # Add a dependency
bun remove <package> # Remove a dependency
bun update           # Update dependencies
bun run <script>     # Run a package.json script
```
